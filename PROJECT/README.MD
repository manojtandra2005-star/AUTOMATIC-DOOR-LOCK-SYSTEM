# Automatic Door Lock System

## Project Overview
The Automatic Door Lock System is a smart security project designed to provide safe and convenient access control. It automatically locks and unlocks the door based on user authentication, reducing the need for traditional keys and improving security.

## Features
- Automatic door locking and unlocking
- Secure user authentication
- Easy to operate
- Low power consumption
- Reliable and cost-effective
- Suitable for homes, offices, and smart buildings

## Components Used
- Arduino Uno
- Servo Motor / Solenoid Lock
- RFID Module / Keypad
- LCD Display
- Buzzer
- LEDs
- Power Supply
- Jumper Wires

## Working Principle
1. The system waits for user authentication.
2. When a valid RFID card or password is detected, the door unlocks.
3. After a few seconds, the door locks automatically.
4. If an invalid card or password is entered, access is denied and an alert is generated.
